Thank You for Downloading Take Notes Desktop!

Follow Below Instructions to Install

Tap the 3 dot menu

On your Chrome or Edge

Open Extensions Tab

Depending on your version, it may be inside submenus

Turn on Developer Mode

Just drag and drop the 'Take Notes Desktop' file to install!

Alternatively...

You may even type chrome://extensions if you are on chrome or edge://extensions if you are on edge in the address bar.

Then enable 'developer mode'

Followed by dragging the downloaded file

Take Notes Desktop will be auto added to your Start Menu/Launchpad.

Enjoy it as usual!
